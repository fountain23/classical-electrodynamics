E and B Fields, Cylindrical Coordinates

A = Magnetic Potential

$$ \vb{A_{\theta}} = - \mu K \left( \left( \frac{ z - \lambda /4 }{ r \sqrt{r^2 + (z-\lambda /4)^2} } \; e^{ -j2\pi \frac{ \sqrt{ r^2 + (z-\lambda /4)^2 }} {\lambda} } \right) - \\\ \left( \frac{ z+\lambda/4 }{ r \sqrt{ r^2+(z+\lambda/4)^2 }} \; e^{ -j2 \pi \frac{ \sqrt{ r^2+(z+\lambda/4)^2)} }{ \lambda} } \right) \right) $$

$$ \mathbf{B_r} = \mu K \left( \left( \frac{r}{ (r^2+(z-\lambda /4)^2)^{1.5} } + j2\pi \frac{ (z- \lambda/4)^2}{ \lambda r(r^2+ (z-\lambda/4)^2) } \right) e^{ -j2 \pi \frac{ \sqrt{ r^2 + (z-\lambda/4)^2 }}{ \lambda }} \\ - \\ \left( \frac{ r } { (r^2+(z+ \lambda/4)^2)^{1.5} } + j2 \pi \frac{ (z+\lambda/4)^2 }{ \lambda r(r^2+(z+\lambda/4)^2) } \right) e^{-j2\pi \frac{ \sqrt{ r^2+(z+\lambda/4)^2}}{\lambda}} \right) $$

$$ \vb{B_z} = \mu K \left[ \left[ (z-\lambda/4) \left( \frac{1}{ (r^2 + (z-\lambda/4)^2 )^{1.5} } + \frac{j2\pi}{ \lambda(r^2+ (z-\lambda/4)^2 ) } \right) e^{ -j2\pi \frac{ \sqrt{r^2+ (z-\lambda /4)^2} }{ \lambda} } \right] - \left[ (z+\lambda/4) \left( \frac{1}{ (r^2+(z+\lambda/4)^2)^{1.5} } + \frac{j2\pi}{ \lambda(r^2+(z+\lambda/4)^2) } \right) e^{-j2\pi \frac{ \sqrt{r^2+(z+\lambda/4)^2}} {λ} } \right] \right] $$

E = Electric Field (cyl)

$$ E = \frac{1}{ j \omega \epsilon \mu } \curl \vb{B} $$

$$ \vb{E_{\theta}} = j2\pi \sqrt{ \frac{\mu}{\epsilon}} \; K \left[ \left( \frac{z-\lambda/4} {\lambda r\sqrt{r^2+ (z-\lambda/4)^2}} \right) e^{ -j2\pi \frac{ \sqrt{r^2+ (z-\lambda/4)^2}}{\lambda}} - \left( \frac{z+\lambda/4}{ \lambda r \sqrt{r^2+(z+\lambda/4)^2} } \right) e^{-j2\pi \frac{ \sqrt{ r^2+(z+\lambda /4)^2}} {λ} } \right] $$

fθ = Osc Frequency

###  E_theta

A = (z - (lambda/4)) / ( lambda*r * sqrt( r**2 + (z - (lambda/4))**2 ))
EA = -I*2*pi * ( sqrt( r**2 + (z - (lambda/4))**2 ) / lambda )

B = (z + (lambda/4)) / (lambda*r * sqrt( r**2 + (z + (lambda/4))**2 ))
EB = -I*2*pi * ( sqrt( r**2 + (z + (lambda/4))**2 ) / lambda )

E_theta = I*2*pi * sqrt(mu/epsilon) * K * (( A * exp(EA) ) - (B * exp(EB)))
